# ICYAGWE Online Shop 🛍️

An elegant and modern e-commerce website for fashion and lifestyle products.

## ✨ Features

- *Beautiful Product Display* - Clean grid layout showing products
- *Shopping Cart* - Add/remove items with real-time updates
- *Responsive Design* - Works perfectly on mobile, tablet, and desktop
- *Dark Mode Support* - Easy on the eyes at night
- *Product Categories* - Organized browsing experience
- *Secure Checkout Interface* - Professional payment page design

## 🛠️ Technologies Used

- *HTML5* - Semantic structure
- *CSS3* - Modern styling with CSS variables
- *JavaScript* - Interactive features
- *Font Awesome 5* - Beautiful icons
- *Google Fonts* - Typography
